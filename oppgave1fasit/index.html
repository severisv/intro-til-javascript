Funksjon i konsollen

  1. Trykk F12
  2. Test konsollen litt ved å skrive litt JavaScript-kode, feks: console.log("hei") og 2 + 2
  3. Lag en funksjon som skriver ut "hallo" og test den:
    function hei() { console.log("hallo") }
    hei()

Manipuler innhold

  1. Hent ut overskriften over:
    var overskrift = document.getElementById("header-2")
    overskrift.innerText = "Ny overskrift"
  2. Hent ut bakgrunnen og gjør den blå med den forhåndsdefinerte funksjonen color
    var bakgrunn = document.getElementById("body")
    color(bakgrunn, "darkBlue")
